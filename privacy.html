Privacy Policy – TT-Poster

TT-Poster respects user privacy and processes data only as required to provide its service.

Data Collected

  • TikTok account authorization tokens
  • Video files uploaded by the user
  • Captions, hashtags, and metadata provided by the user

Purpose of Processing

Data is used solely to enable publishing videos to TikTok via the Content Posting API.

Data Sharing

Data is not sold or shared with third parties, except TikTok as required for publishing.

Data Retention

Data is stored only as long as necessary to provide the service.
